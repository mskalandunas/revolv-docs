Bells and Whistles

This second example is a bit more garnished. It has titles under the images, and allows the carousel to be clickable with links corresponding to each image. If you'd like to use one link to represent all images in the carousel, I'd recommend just wrapping the image in an anchor rather than using a link property. Clicking these images and titles will open a trailer in a new tab.
